Feeds Module

Description

StreetHawk's Feed module is used for sending feeds to your application.

Accessing the Feeds Module

To access this module from JavaScript, you would do the following:

var shFeed = require("com.streethawk.Feeds");

The shFeed variable is a reference to the Module object.

Reference

https://streethawk.freshdesk.com/solution/articles/12000000944-feed

feeds.function

  • Callback for new feed available

    function newFeedCallback(e) { alert("Callback for new feed available."); } //Register new feed callback to StreetHawk SDK shFeed.notifyNewFeedCallback({newfeed: newFeedCallback}); //must use key "newfeed"

  • Callback for fetching feed items

    function fetchFeedCallback(e) { Ti.API.info("Fetch feed items: " + JSON.stringify(e)); } //Send request to fetch feed items. shFeed.shGetFeedDataFromServer(, {feed: fetchFeedCallback}); //must use key "feed"

  • Send request for feed ack

    shFeed.reportFeedAck();

  • Send feed result

    shFeed.reportFeedRead(, );
